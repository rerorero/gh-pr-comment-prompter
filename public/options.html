PR Comment Prompter

Configure the prompt template copied to your clipboard when clicking the button on GitHub PR comments.

Available variables

{{comment}} The full text of the PR comment {{filePath}} File path (inline review comments only) {{lineNumber}} Line number in the file (inline review comments only) {{branchName}} Head branch name of the PR {{prNumber}} Pull request number {{owner}} Repository owner (user or organization) {{repo}} Repository name

Variables without a value (e.g. {{filePath}} on a general PR comment) are left as-is in the output.
